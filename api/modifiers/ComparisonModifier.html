# ComparisonModifier

# ComparisonModifier ⇐ Modifier

A ComparisonModifier is the base modifier class for comparing values.

Abstract class

This is meant as an abstract class and should not be used directly. You can use one of the extended modifiers, or extend the class yourself.

Kind: global abstract class
Extends: Modifier
